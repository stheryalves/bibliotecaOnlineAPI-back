Tecnologias utilizadas:

mongodb

clone esse repositório $ git clone https://github.com/stherystyll/bibliotecaOnlineAPI-back.git

acesse a pasta do projeto - 
$ cd <repo_name>

instale as dependências - 
$ npm install

abra outro terminal e rode o projeto - 
$ npm run dev
